# Economics Blog Website

একটি সহজ এবং মোবাইল-রেসপনসিভ ইকোনমিক্স ব্লগ ওয়েবসাইট, যেখানে অর্থনীতির মৌলিক ধারণা ও নোটগুলো সুন্দরভাবে উপস্থাপন করা হয়েছে। (A simple and mobile-responsive economics blog website featuring basic economic notes and concepts.)

---

## 🚀 ফিচারসমূহ (Features)
* **মোবাইল রেসপনসিভ:** যেকোনো ডিভাইস (মোবাইল, ট্যাবলেট, ডেস্কটপ) থেকে সহজে ব্রাউজ করা যায়।
* **সহজ নেভিগেশন:** ব্যবহারকারীরা সহজেই বিভিন্ন ইকোনমিক্স নোট ও কনসেপ্ট খুঁজে পাবেন।
* **ইউজার-ফ্রেন্ডলি ডিজাইন:** পড়ার সুবিধার্থে পরিষ্কার এবং দৃষ্টিনন্দন লেআউট।

## 🛠️ যা যা ব্যবহার করা হয়েছে (Tech Stack)
* **HTML5:** ওয়েবসাইটের মূল স্ট্রাকচার বা কাঠামোর জন্য।
* **CSS3:** ওয়েবসাইটটিকে আকর্ষণীয় ও রেসপনসিভ করার জন্য।
